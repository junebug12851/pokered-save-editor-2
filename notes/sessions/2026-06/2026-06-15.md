Two offscreen gotchas found + fixed (now in `reference/screenshots.md` and `reference/qt-patterns.md`):

1. **Tofu text.** The offscreen platform's FreeType font DB finds no fonts unless pointed at one, so
   all UI text rendered as boxes (the in-game *bitmap* name preview still rendered via the font image
   provider, which masked it at first). Fix: the tool sets `QT_QPA_FONTDIR` to the OS font dir when
   empty (`%WINDIR%\Fonts` on Windows).
2. **Blank grabs.** Forcing `QT_QUICK_BACKEND=software` makes offscreen `grabWindow()` produce real
   pixels with no GPU (so it also works in CI/Docker); trade-off is `MultiEffect` shadows/desaturation
   are skipped â€” acceptable for documentation shots.
